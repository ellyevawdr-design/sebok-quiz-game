SE-BoK Quiz Game
Created by: ELLYSIA EVA ANAK WINSTON (82538)
For: SEB Contract & Procurement Governance (SE-BoK AI Awareness)

A full-stack client-server quiz web application evaluating software engineering prompt qualities.

Architecture Stack
Frontend: HTML5, Tailwind CSS, JavaScript (Fetch API / Asynchronous DOM updates)
Backend: Node.js, Express.js REST API microservice
Data Store: Local JSON system (`database.json`)

How to Run Locally
1. Clone the repository.
2. Open the `backend` folder, run `npm install`, and start the server with `node server.js`.
3. Open the `frontend` folder using a local server environment (e.g., VS Code Live Server or `npx serve`).
